Close-Reading Strategy

Download PDF

Students will read multiple documents in this digital inquiry book. This is a suggested reading strategy for teachers to support students in understanding and analyzing documents. We’ve included this reading strategy with differentiated options in the teacher instructions for Chapter 2, Chapter 3, and Chapter 4

How to Model the Close-Reading Strategy  

  1. Read the document out loud for students, modeling identifying unknown vocabulary. 
  2. Go through embedded vocabulary, taking time to read the definitions.  
  3. Model pondering: Based on this reading, what do I think this document is about? 
  4. Read the document’s comprehension question(s). 
  5. Read the paraphrased version of the document, noting that this is a helpful tool for understanding, but is not the document itself. 
  6. Read the document out loud for a second time. 
  7. Model answering the comprehension question(s) out loud; use the class as a “thinking partner.” 
  8. After discussing responses with the class, write the response in notes. 

Differentiated Close-Reading Strategies

Use these differentiated options with the document banks in Chapter 2, Chapter 3, and Chapter 4. 

Supported Option: Direct Instruction leading to Guided Practice

Direct Instruction

Model close-reading of Document A (see How to Model the Close-Reading Strategy).

Guided Practice

Transition to guided practice for Documents B and C:

  1. Read Document B aloud with students.
  2. Ask students to identify unknown words and review any embedded or new vocabulary.
  3. Ask students: Based on this first read, what do you think the document is about?
  4. Check for misconceptions.
  5. Read the comprehension question.
  6. Read the paraphrased version of Document B.
  7. Read Document B for a second time.
  8. Solicit responses to the comprehension question for Document B to guide student understanding. Do this as a whole class, or as a Think, Pair, Share activity.
  9. Note student responses or have students record each other's responses for Document B.
  10. Release students in pairs or small groups to respond to the formative assessment entitled Document Analysis, which follows each document.
  11. Repeat above steps with Document C.
  12. Return to whole-class discussion before students respond to the chapter’s Guiding Question.

Independent Option: Guided Practice leading to Independent Work

Guided Practice

As a class, use Guided Practice to close read Document A (see Supported Option: Direct Instruction leading to Guided Practice).

Independent Work

Release students to independent practice for Documents B and C:

  1. Students read Document B.
  2. Students identify unknown vocabulary and review embedded vocabulary.
  3. Turn and Talk: Based on this first read, what do students think this document is about?
  4. Students read the comprehension question.
  5. (Optional) Students read Document B paraphrase.
  6. Students read Document B for the second time.
  7. Students respond to the Document B comprehension question in pairs or small groups.
  8. Students respond to the formative assessment entitled Document Analysis, which follows each document.
  9. Repeat above steps for Document C.
  10. (Optional) Students discuss texts in small groups or as a whole class before responding to the chapter’s Guiding Question.
